General description

 

RootEdu Education Institute is a fully responsive premium HTML5 template build with Bootstrap framework. It is designed for education, online courses, training centers, academy colleges, schools, universities and other education related websites. RootEdu Education Institute has everything you need to create an excellent education websites. It comes with concise 14 HTML pages with everything you need (f.e. blog, courses, events, teacher profile and newsletter section and much more) and a potential to grow with more in the future. Clear and trendy designed, built with LESS, it looks perfect on all major browsers, tablets and phones. Color scheme and fonts can be easy customized as you want. Feel free to visit the preview link or contact us with a feature request.

Instructions

General structure

●     /app - source files (non-minified assets, source LESS files)

○     /assets/bootstrap - original Bootstrap files

○     /assets/css - compiled CSS files

○     /assets/fonts - fonts

○     /assets/ico - favicon images

○     /assets/img - image assets

○     /assets/video - video assets

○     /assets/js - JS scripts

○     /assets/less -source LESS styles

○     /assets/plugins - plugins used throughout the theme

○     /assets/documentation - template documentation

●     /dist - ready for production files. This is the folder that you need if you don’t want to work on the source files.

Files compilation

If you prefer to work on the source LESS files, please follow the steps below:

1.    Open a command line from the /admin folder.

2.    Run npm install to install required dependencies (taken that you have node.js installed on your machine).

3.    Once everything is installed, run one of the commands below. Available commands:

a.    gulp - to make an initial compilation and start watching for changes in LESS files. This will also start a local server and a locahost:3000 page you will be able to preview changes in.

b.    gulp build - to compile files for production. This will compile LESS files to CSS and minify assets.

 

Color schemes

If you need a different color scheme, you need to modify source LESS files. Please, follow the steps below:

●     Perform the steps, described in p. "File compilation".

●     Open to /app/assets/less/components/colors.less in your favourite text editor.

●     Change the value of the @brand-primary variable to your preferred color.

●     Save the file and run gulp build to build the /dist folder again.

 

Photos

All photos in the preview are not included into the download file. Please follow the steps

below in order to replace them with your custom images:

●     index.html

Please find an <img> tag that you need and replace its “src” attribute content with an actual link to your image (e.g. <img class="img-responsive" src="assets/img/your_photo.jpg" alt="...">).

●   styles.css

     Please find a selector that you need and replace its “url” value content with an actual link to your image (e.g. background: url(“../img/your_photo.jpg”); ).

 

Contact form

This template contains a fully functioning PHP contact form.

Note: The contact form will not work in your local environment without a server that supports PHP.

In order to set up the contact form, please follow the steps below:

1. Open the /app/assets/plugins/contact/config.php file and fill out the required information:

 - Your email address ($to_email)

 This is an email address new emails will be sent to.

 - Email subject ($mail_subject)

 This is a subject of new emails you will receive.

2. Save config.php file.
